# Récupération et installation

```bash
> git clone https://git.meteo.fr/cnrm-cen/hugo_collab/pandas-basics
> cd pandas-basics
> virtualenv -p python3 venv # en option pour travailler dans un env de travail local - sinon se placer dans son environnement de travail préféré
> source venv/bin/activate # idem en option
> pip install -r requirements.txt
> jupyter notebook # c'est parti
```
